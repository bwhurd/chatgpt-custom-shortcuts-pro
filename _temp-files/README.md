# Temporary Files

This folder holds scratch artifacts, copied inspector captures, reference scripts, logs, and other support material that should stay outside the default startup path.

- Do not read or search this folder unless the current user request explicitly names a file or path here.
- Keep durable project rules in `AGENTS.md`, durable subsystem detail in `SPECS/`, and active or deferred work in `PLANS/`.
- Keep this README visible; other contents are ignored by default.
